# Nvidia Nemotron Competition Submission - CMPE 258 Homework 3

This repository contains my solution for the Nvidia Nemotron competition on Kaggle, which also serves as Homework 3 for my Deep Learning course (CMPE 258) at San Jose State University.

## Project Overview

The objective of this project is to build and evaluate models for the Nvidia Nemotron Kaggle competition. The implementation includes the complete machine learning pipeline, from data preprocessing and model training to evaluation and inference.

## Repository Structure

- `nvidia-nemotron-submission.ipynb`: The main Jupyter Notebook containing the full implementation of the solution. This notebook covers:
  - Data loading and exploration
  - Preprocessing and feature engineering
  - Model architecture definition and training
  - Evaluation metrics and analysis
  - Generating predictions for submission

This solution is designed for the Nvidia Nemotron competition on Kaggle. The goal is to leverage deep learning techniques to achieve high performance on the competition's evaluation metric.

## Usage

To run the notebook:
1. Ensure you have a suitable environment with Jupyter Notebook and necessary deep learning libraries (e.g., PyTorch or TensorFlow) installed.
2. Download the dataset from the Kaggle competition page.
3. Open `nvidia-nemotron-submission.ipynb` and update any file paths if necessary.
4. Run the cells sequentially to train the model and generate the submission file.
